# ИИ Сонник - Запуск через командную строку
Запрещается использование в комерческих целях, данный проект пренадлежит ttagerr.

## Предварительные требования

- **Node.js** 16.x или выше
- **npm** (обычно устанавливается с Node.js)
- **База данных PostgreSQL**

## Установка и запуск

### 1. Установка зависимостей

```cmd
cd путь_к_проекту
npm install 
```
### 2. Редактирование ключей

В файле fixed-server.js заменить API GigaChata на свой в 12 строке

### З. Запуск

```cmd
cd путь_к_папке_backend
node fixed-server.js
```
